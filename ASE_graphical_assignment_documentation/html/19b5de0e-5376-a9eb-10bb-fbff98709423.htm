      Remarks

      The method supports the following custom commands:
      • `tri` - Creates an instance of AppTriangle for drawing triangles.
      • `reset` - Creates an instance of AppReset for resetting the canvas.
      • `write` - Creates an instance of AppWrite for writing text.
      • `write` - Creates an instance of AppClear for clearing canvas.
      • `circle` - Creates an instance of AppCircle for drawing circles.
      • `rect` - Creates an instance of AppRectangle for drawing rectangles.
      • `int` - Creates an instance of AppInt for handling integer operations.
      • `real` - Creates an instance of AppReal for handling restrictions of real number operations.
      • `array` - Creates an instance of AppArray for handling restrictions of arrays.
      • `if` - Creates an instance of AppIf for handling restrictions of if conditional statements.
      • `end` - Creates an instance of AppEnd for handling restrictions of the end of a block.
      • `method` - Creates an instance of AppMethod for handling restrictions of method definitions.
      • `for` - Creates an instance of AppFor for handling restrictions of for looping operations.
      • `while` - Creates an instance of AppWhile for handling restrictions of while loops.
      • `else` - Creates an instance of AppElse for handling restrictions else conditions.
      • `call` - Creates an instance of AppCall for handling restrictions of function calls.
      If the command type is not recognized, it calls the base class.

      Exceptions

      FactoryExceptionThrown when the input command type is invalid or it does not belong to the command type defined here.
