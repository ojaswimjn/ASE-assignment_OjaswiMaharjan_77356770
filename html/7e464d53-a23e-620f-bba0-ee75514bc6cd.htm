      Classes

      AppArray Removes the restrictions made by the base Array class. The restriction was enabled to run only 2 'array' based commands.
      AppCall Reduces the counter value by 1 to remove restrictions made by the base Call class. The restriction was enabled to run only 2 compound based commands.
      AppCanvas A class which provides methods for graphical operations such as drawing diffrent shapes, lines in different canvas position. Implements ICanvas interface.
      AppCircle Circle Command to draw on the canvas with the fill functionality.
      AppClear Represents a command to clear the canvas. It inherits from the CanvasCommand class.
      AppCommandFactory A Command factory class for making new commands. It is used to create object of a new class for the executoin of the new commands. Inherits from the CommandFactorybase class. It is the existing base class which creates new commands.
      AppElse Inherits from the base Else class.
      AppEnd Inherits from the base End class.
      AppFor Provides functionality for relaxing restrictions of "for" loops.
      AppIf Removes the restrictions made by the base CompoundCommand class. The restriction was enabled to run only 2 'if' statements in the user-defined commands.
      AppInt Removes the restrictions made by the base Int class. The restriction was enabled to run only 5 'int' based commands.
      AppMethod Removes the restrictions made by the base CompoundCommand class. The restriction was enabled to run only 2 'method' statements in the user-defined commands.
      AppReal Removes the restrictions made by the base Real class from the BOOSE library. The restriction was enabled to run only 5 'real' based commands.
      AppRectangle Rectangle command to draw on the canvas with the fill functionality. Inherits from the Rect class.
      AppReset Represents a command to reset the canvas to its original position. It inherits from the CanvasCommand class.
      AppStoredProgram Represents a stored program that executes commands in sequence by removing the cycle restrictions. Inherits from the StoredProgram base class.
      AppTriangle A class to draw a triangle on the canvas. Inherits from the CommandOneParameter base class.
      AppWhile Removes the restrictions made by the base CompoundCommand class. The restriction was enabled to run only 2 'while' statements int the user-defined commands.
      AppWrite Writes the text to the canvas command in the specific position.
      CommandLogger Log the user typed commands on the screen as history in a textfile automatically. It uses singleton design pattern. It ensures that only one instance of CommandLogger exists and provides methods for adding, retrieving, and clearing commands.
      Form1 Main form for the application. It takes the commands from the user via a text box. On the basis of the user input command graphical objects is drawn onto the picturrebox. When error is encountered during program execution, a errror message is displayed in the text box with detail error information.
